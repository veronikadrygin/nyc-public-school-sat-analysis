# NYC Public School SAT Analysis

## Project Overview

This project analyzes SAT performance across public schools in New York City.

The goal is to identify top-performing schools, compare academic performance, and examine differences between boroughs using SAT test results.

## Business Questions

1. Which schools have the best math results?
2. Which schools have the highest overall SAT scores?
3. Which borough shows the greatest variation in SAT performance?

## Tools Used

- Python
- Pandas

## Analysis

The analysis focused on three key tasks:

- Identifying schools with exceptional math performance
- Ranking schools by total SAT score
- Comparing SAT score variability across New York City boroughs

A combined SAT score was calculated using:

- Average Math Score
- Average Reading Score
- Average Writing Score

## Key Findings

- Several schools achieved outstanding mathematics results, scoring at least 80% of the maximum possible SAT math score.
- The top-performing schools were identified based on total SAT scores across all sections.
- SAT performance varied significantly across boroughs, with one borough showing notably greater score dispersion than others.
